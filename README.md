# Simple Music Player developed with vanilla web technology 🎵
A simple music player for the web that can read the metadata of the uploaded files.
This project started as a university assignment, i decided to refine and publish like my first GitHub repository to document my learning journey.

## Features 📝
- Reproduce uploaded audio from local files.
- Play / Pause functionality.
- Track navigation.
- Metadata extraction (name, artist, picture).

## Technologies 👨‍💻
- HTML.
- CSS.
- JavaScript.
